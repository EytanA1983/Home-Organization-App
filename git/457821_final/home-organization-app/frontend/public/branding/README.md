Place branding image assets in this folder with these exact names:

- logo.png
- room-bg.png
- course-cover.png

They are referenced by:

- /branding/logo.png
- /branding/room-bg.png
- /branding/course-cover.png

After replacing `logo.png`, you can crop away near-white margins with:

`npm run branding:trim-logo` (from `frontend/`)
