# הפעלת Frontend - הוראות מהירות

## הבעיה:
אתה נמצא ב-`C:\Users\maore\frontend` במקום בתיקיית הפרויקט.

## הפתרון:

### 1. עבור לתיקייה הנכונה:
```powershell
cd "C:\Users\maore\git\סידור וארגון הבית - אלי מאור\frontend"
```

### 2. התקן תלויות (אם צריך):
```powershell
npm install
```

### 3. הרץ את השרת:
```powershell
npm run dev
```

### 4. פתח בדפדפן:
**Development (ללא Docker):**
http://localhost:5173

**Production (עם Docker):**
http://localhost:3000

---

## או השתמש בסקריפט:
לחץ כפול על `RUN-FRONTEND.ps1` בתיקיית הפרויקט הראשית.
