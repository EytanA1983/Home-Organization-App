## ⚠️ קבצים שצריך לעדכן (אופציונלי)

הקבצים הבאים עדיין מזכירים 5173, אבל הם לא קריטיים:
- קבצי תיעוד (README.md, HOW-TO-RUN.md, וכו')
- קבצי סקריפטים ישנים (RUN-FRONTEND.ps1, RUN-SERVERS.ps1, וכו')

**הערה**: אם אתה משתמש בקבצים האלה, עדכן אותם ידנית או השתמש ב-`start-app.ps1` המעודכן.

## 🚀 איך להריץ עכשיו?

### Development:
```powershell
.\start-app.ps1
# או ידנית:
cd frontend
npm run dev
# פתח: http://localhost:5178
```

### Production:
```powershell
docker compose up
# פתח: http://localhost:3000
```
