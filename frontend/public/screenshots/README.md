# PWA Screenshots

Place screenshots here for the PWA install prompt:

1. `home-desktop.png` - 1280x720 (desktop screenshot)
2. `home-mobile.png` - 750x1334 (mobile screenshot)

These screenshots are shown when users install the PWA on supported browsers.

## How to Generate

1. Run the app in development mode
2. Take screenshots of the home page
3. Save them with the correct dimensions
4. Reference them in `manifest.json`
