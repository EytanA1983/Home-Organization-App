# i18n (Internationalization) Documentation

## Overview

המערכת כוללת תמיכה מלאה בריבוי שפות עם react-i18next:
- **עברית** (ברירת מחדל)
- **אנגלית**
- **רוסית**

## Installation

התלויות כבר מותקנות ב-`package.json`:
- `i18next` - Core i18n library
- `react-i18next` - React integration
- `i18next-browser-languagedetector` - Auto language detection

## Configuration

### i18n Config (`src/i18n/config.ts`)

הקובץ מגדיר:
- שפות נתמכות: `he`, `en`, `ru`
- שפת ברירת מחדל: `he`
- Namespaces: `common`, `rooms`, `tasks`, `auth`, `settings`
- Auto-detection: localStorage, navigator, htmlTag

## Translation Files

### Structure

```
src/i18n/locales/
  ├── he.json (עברית)
  ├── en.json (English)
  └── ru.json (Русский)
```

### Namespaces

כל קובץ תרגום כולל namespaces:
- `common` - טקסטים כלליים
- `rooms` - טקסטים הקשורים לחדרים
- `tasks` - טקסטים הקשורים למשימות
- `auth` - טקסטים הקשורים לאימות
- `settings` - טקסטים הקשורים להגדרות

## Usage

### Basic Translation

```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { t } = useTranslation();
  
  return <h1>{t('common.app_name')}</h1>;
}
```

### With Namespace

```typescript
import { useTranslation } from 'react-i18next';

function RoomsList() {
  const { t } = useTranslation('rooms');
  
  return <h2>{t('title')}</h2>;
}
```

### With Variables

```typescript
const { t } = useTranslation('rooms');

// In translation file: "tasks_count": "{{count}} משימות"
<p>{t('tasks_count', { count: 5 })}</p>
```

### Pluralization

```typescript
// In translation file:
// "tasks_count_one": "משימה אחת"
// "tasks_count_other": "{{count}} משימות"

<p>{t('tasks_count', { count: 1 })}</p> // "משימה אחת"
<p>{t('tasks_count', { count: 5 })}</p> // "5 משימות"
```

## Language Switcher

### Component

```typescript
import { LanguageSwitcher } from './components/LanguageSwitcher';

function App() {
  return (
    <div>
      <LanguageSwitcher />
      {/* ... */}
    </div>
  );
}
```

### Manual Language Change

```typescript
import { useTranslation } from 'react-i18next';

function MyComponent() {
  const { i18n } = useTranslation();
  
  const changeLanguage = (lng: string) => {
    i18n.changeLanguage(lng);
    localStorage.setItem('i18nextLng', lng);
  };
  
  return (
    <button onClick={() => changeLanguage('en')}>
      English
    </button>
  );
}
```

## Adding New Translations

### 1. Add to Translation Files

**he.json:**
```json
{
  "common": {
    "new_key": "ערך חדש בעברית"
  }
}
```

**en.json:**
```json
{
  "common": {
    "new_key": "New value in English"
  }
}
```

**ru.json:**
```json
{
  "common": {
    "new_key": "Новое значение на русском"
  }
}
```

### 2. Use in Component

```typescript
const { t } = useTranslation('common');
<p>{t('new_key')}</p>
```

## RTL Support

עברית ורוסית תומכות ב-RTL (Right-to-Left):

```css
/* In index.css or component styles */
[dir="rtl"] {
  direction: rtl;
  text-align: right;
}
```

## Best Practices

1. **Use Namespaces** - ארגן תרגומים לפי namespaces
2. **Consistent Keys** - השתמש במפתחות עקביים
3. **Variables** - השתמש במשתנים לתרגומים דינמיים
4. **Pluralization** - השתמש ב-pluralization למספרים
5. **Default Language** - תמיד שמור תרגום בעברית

## Testing

### Change Language in Browser

```javascript
// In browser console
localStorage.setItem('i18nextLng', 'en');
location.reload();
```

### Check Current Language

```typescript
const { i18n } = useTranslation();
console.log(i18n.language); // 'he', 'en', or 'ru'
```
